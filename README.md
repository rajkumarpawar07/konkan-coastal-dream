# 🌊 Konkan Coastal Dream

> A beautifully crafted **6-Day Konkan Road Trip Itinerary** web app — your digital travel companion for exploring Maharashtra's breathtaking coastal paradise.

🔗 **Live Demo:** [konkantrip.vercel.app](https://konkantrip.vercel.app)

---

## 📖 About

**Konkan Coastal Dream** is an interactive travel itinerary web application that guides travelers through a curated 6-day road trip along the Konkan coast of Maharashtra. From pristine beaches and ancient forts to lush green Sahyadri foothills and seafood havens, this app makes trip planning immersive and effortless.

Whether you're a weekend wanderer or a seasoned road tripper, this project serves as a visually rich, day-by-day travel guide for one of India's most scenic coastal routes.

---

## ✨ Features

- 🗓️ **Day-by-Day Itinerary** — Structured 6-day travel plan with highlights for each stop
- 🏖️ **Coastal Destinations** — Curated beaches, waterfalls, forts, and cultural spots
- 📱 **Responsive Design** — Fully mobile-friendly for on-the-go trip planning
- 🎨 **Modern UI** — Clean, elegant interface built with shadcn/ui and Tailwind CSS
- ⚡ **Blazing Fast** — Powered by Vite for instant page loads
- 🔷 **Type-Safe** — Built entirely in TypeScript for a robust codebase

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm / bun

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/rajkumarpawar07/konkan-coastal-dream.git

# 2. Navigate into the project directory
cd konkan-coastal-dream

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be available at `http://localhost:5173` by default.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 🌍 Deployment

The project is deployed on **Vercel**. Any push to the `main` branch triggers an automatic redeployment.

To deploy your own fork:

1. Fork this repository
2. Import it into [Vercel](https://vercel.com/)
3. Set the framework preset to **Vite**
4. Hit **Deploy** 🚀

## 🗺️ The Konkan Route

The itinerary covers some of Maharashtra's most beloved destinations along NH-66:

- 🏰 **Murud-Janjira** — The legendary sea fort
- 🌊 **Ganpatipule** — Sacred beach town
- 🌿 **Ratnagiri** — The Alphonso mango capital
- 🐚 **Tarkarli** — Crystal-clear waters & scuba diving
- 🌴 **Malvan** — Seafood & Sindhudurg Fort
- 🌅 **Vengurla** — Tranquil beaches at the Goa border
